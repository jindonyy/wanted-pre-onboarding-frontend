# 원티드 프리온보딩 프론트엔드 과제

## 프로젝트 실행 방법

1. root 경로에 .env 파일을 생성한 뒤 아래의 내용을 작성한다.

```
REACT_APP_BASE_URL=(주어진 API 주소)
```

2. 터미널에 `npm install` 와 `npm start`를 입력한다.

```
npm install
npm start
```

3. [http://localhost:3000/](http://localhost:3000/) 에 접속한다.
